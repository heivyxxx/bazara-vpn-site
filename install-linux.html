Установка BazaraVPN на Linux

  1. Скачай Hiddify Next для Linux с официального GitHub.
  2. Распакуй архив и запусти программу (обычно Hiddify или Hiddify.AppImage).
  3. Скопируй свою подписку BazaraVPN и добавь её через «Новый профиль» → «Добавить профиль из буфера обмена».
  4. В настройках выбери режим VPN или Системный прокси (только для браузера).
  5. Выбери сервер, подключайся — и всё, ты под защитой!
